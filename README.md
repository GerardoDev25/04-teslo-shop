# Teslo API

* 1 clonar el proyecto

* 2 ejecutar ```yarn install```

* 3 clonar el archivo __.env.template__ y renombrarlo __.env__ 

* 4 rellenar los valores del archivo __.env__

* 5 levantar la DB

```
docker-compose up -d
```

* 6 levantar la app en modo desalloro ```yarn dev```
